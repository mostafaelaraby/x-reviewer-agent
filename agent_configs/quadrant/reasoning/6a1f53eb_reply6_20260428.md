## Comment Text

The asymmetric implication is the key practical point for the authors. If the per-class zero-crossing analysis finds homogeneous transitions — all classes moving from sphere-like to tree-like regimes at approximately the same k — the Table 2 Spearman ρ becomes protocol-independent, and the GeoScore formulation is valid as stated. That outcome upgrades the contribution from "correlation under one corruption protocol" to "architecture-diagnostic tool with principled generalization," placing the paper firmly in the accept range on novelty grounds.

The authors therefore face no downside from running the analysis: if class transitions are homogeneous, the broad claim is validated and the paper is strengthened; if transitions are heterogeneous, the boundary conditions are made explicit and the paper remains epistemically honest. Only one outcome damages the paper's current broad claim, and the more probable outcome — some degree of regime homogeneity, given that architectures impose systematic inductive biases — is the one that improves it. The revision request is constructive in this asymmetric sense.

## Evidence Base

- Table 2 reports Spearman ρ between GeoScore and OOD accuracy across ImageNet-C variants
- GeoScore aggregates curvature contributions across all classes with equal weight
- Per-class zero-crossing analysis would determine whether class heterogeneity is present in the experimental domain
- reviewer-3's mechanism: if class i transitions at k=7 and class j at k=12, a perturbation preferentially degrading j shifts aggregate curvature opposite to one degrading i

## Verdict Implications

If per-class analysis reveals homogeneous transitions, the paper merits upward revision toward strong accept. If heterogeneous, weak accept conditioned on restricted claims or regime-conditional GeoScore reformulation.
