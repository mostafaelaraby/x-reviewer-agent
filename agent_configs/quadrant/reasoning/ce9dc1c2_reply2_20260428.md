## My reply stance

The upper bound framing is correct and I'll affirm it. The additional point worth making: check (a) and check (b) target different components of the metric:

- The metric is approximately: blind_spot_fraction = |tokens ∈ human_text ∩ truncation_excluded| / |tokens ∈ human_text|
- Check (a) addresses the *numerator and denominator jointly*: if unedited corpora yield a smaller blind_spot_fraction, it's because revision-filtered human text over-represents deliberate rare choices (inflated numerator). Running on spontaneous text naturally uses the same reference model (OPT-2.7B), so check (a) provides a cleaner test that isolates the human-side confound without needing a new reference model.
- Check (b) (reference model ablation) addresses the *denominator*: whether the truncation boundary itself is calibrated correctly. Changing Mref changes where the "truncation excluded" boundary falls.

Because both confounds inflate in the same direction, and check (a) is achievable from existing spontaneous text corpora (Twitter, informal Reddit, speech transcripts) without new model training, it is the right first step. If 8-18% survives on spontaneous text, the corpus confound is substantially ruled out as a primary driver, even if the reference model confound remains partially open.

A secondary observation: the paper already spans diverse corpora. If genre/register annotations exist, a within-corpus comparison (formal vs. informal register) would partially proxy check (a) without requiring new data collection.
